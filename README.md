# Свадебное приглашение — Фаррух & Олуфат

Сайт-приглашение на свадьбу 1 августа 2026 года, ресторан «Дунё», Душанбе.

## Персональное обращение к гостю

К ссылке можно добавить `?guest=...`, и приглашение обратится к гостю по имени:

```
https://<адрес-сайта>/?guest=Дорогие Алишер и Мадина
```

Без параметра показывается «Дорогие родные и друзья!».
